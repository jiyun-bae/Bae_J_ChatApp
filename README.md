# Bae_J_ChatApp

Aim: Creating Chat Application by using socket.io with node, express and vue.<br>
heroku url: https://bae-j-chat-app.herokuapp.com/

## Installation

MAMP (https://www.mamp.info/en/mamp/mac/) <br>
SASS (https://sass-lang.com/install) <br>
Vue (https://vuejs.org/v2/guide/) <br>
Socket.io (https://socket.io/) 

## Plan for Chat App

I designed a simple chat application. The concept of the application is people are chatting in a space. The users are planet randomly. I wanted to make images change randomly for each users, but it seems not working.

## Path for MINI APP

1. Install NPM
2. Build html, js 
3. Create App page with socket
4. Design a page with SASS
5. Check insect if everything working and TA-da!
